📌 Sobre o projeto

Este projeto é um portfólio online desenvolvido com o objetivo de apresentar minhas habilidades, projetos e evolução ao longo da minha jornada na área de tecnologia.

A aplicação reúne projetos desenvolvidos durante meus estudos, além de destacar as principais tecnologias que venho utilizando no desenvolvimento front-end.

🚀 Acesse o projeto online

🔗 https://kellyalvesdev.github.io/

🛠️ Tecnologias utilizadas
React
JavaScript
HTML5
CSS3
Git & GitHub


📱 Funcionalidades

Navegação entre seções (Sobre, Projetos, Habilidades e Contato)
Exibição de projetos com tecnologias utilizadas
Layout responsivo para diferentes dispositivos
Interface moderna e intuitiva

⚙️ Como rodar o projeto localmente
# Clone o repositório
git clone https://github.com/KellyalvesDev/kellyalvesdev.github.io.git

# Acesse a pasta
cd kellyalvesdev.github.io

# Instale as dependências
npm install

# Rode o projeto
npm start

🌐 Deploy

O projeto foi publicado utilizando o GitHub Pages.

Para realizar o deploy:

npm run build
npm run deploy
