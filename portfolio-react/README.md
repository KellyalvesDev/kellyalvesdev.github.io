# 💻 Portfólio - Kelly Alves

## 📌 Sobre o projeto

Este projeto é um portfólio online desenvolvido para apresentar minhas habilidades, projetos e evolução na área de tecnologia.

A aplicação reúne projetos desenvolvidos durante meus estudos, destacando as principais tecnologias que utilizo no desenvolvimento front-end.

## 🚀 Acesse o projeto online

🔗 [https://kellyalvesdev.github.io/](https://kellyalvesdev.github.io/)

## 🛠️ Tecnologias utilizadas

- React
- JavaScript
- HTML5
- CSS3
- Git & GitHub

## 📱 Funcionalidades

- Navegação entre seções (Sobre, Projetos, Habilidades e Contato)
- Exibição de projetos com tecnologias utilizadas
- Layout responsivo para diferentes dispositivos
- Interface moderna e intuitiva

## ⚙️ Como rodar o projeto localmente

```bash
# Clone o repositório
git clone https://github.com/KellyalvesDev/kellyalvesdev.github.io.git

# Acesse a pasta
cd kellyalvesdev.github.io

# Instale as dependências
npm install

# Rode o projeto
npm start
```

## 🌐 Deploy

Publicado com **GitHub Pages**.

```bash
npm run build
npm run deploy
```

## 🗂️ Estrutura do Projeto

```
src/
├── main.jsx              # Ponto de entrada da aplicação
├── App.jsx               # Componente raiz — monta todos os componentes
├── index.css             # Estilos globais e variáveis CSS
└── components/
    ├── Nav.jsx           # Barra de navegação fixa com scroll effect
    ├── Nav.css
    ├── Hero.jsx          # Seção principal com foto e apresentação
    ├── Hero.css
    ├── Sobre.jsx         # Seção "Quem sou eu" com links sociais
    ├── Sobre.css
    ├── Skills.jsx        # Grid de habilidades técnicas
    ├── Skills.css
    ├── Projetos.jsx      # Galeria de projetos com filtro por categoria
    ├── Projetos.css
    ├── Contato.jsx       # Formulário de contato com envio via WhatsApp
    ├── Contato.css
    ├── Footer.jsx        # Rodapé com links sociais
    └── Footer.css
```

## 🚀 Como rodar localmente

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Gerar build de produção
npm run build
```

## 📦 Como publicar no GitHub Pages

```bash
# Instalar gh-pages (já está no package.json)
npm install

# Gerar build e publicar
npm run build
npx gh-pages -d dist
```
